Documentation Page for Movie API.

The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Request URL HTTP Method URL parameters Request body data format Respond body data format
Welcome message. / GET None A welcome text message.
Get a list of all movies. /movies GET None A JSON object holding data about all movies.
Get data (description, genre, director) about a single movie by title. /movies/[Title] GET /movies/:title None A JSON object holding data about a single movie containing a title, description, genre, director.

Example:
{ "title": "Amadeus", "description": "The film follows a fictional rivalry between Mozart and Italian composer Antonio Salieri at the court of Emperor Joseph II", "director": "Milos Forman", "genres": [ " Biography", "Drama", "History", "Music" ],}
Get data about a movie genre by name. /movies/genres/[Genre] GET /movies/genres/:genres None A JSON object holding data about the genres.
Get data about a movie director by name. /movies/directors/[Name] GET /movies/directors/:name None A JSON object holding data about the director.
Register new user. /users POST A JSON object holding data about the user to create.
A JSON object holding data about the user that was added.
Update a user information. /users/[Username] PUT /users/:username a JSON object holding data about the user to update.
A text message indicating the info that was updated.
Add a movie to Favourites. /users/[Username]/favourites/[MovieTitle] POST /users/:username/favourites/:title None A text message indicating whether the movies was successfully added to favourites.
Remove movie from Favourites. /users/[Username]/favourites/[MovieTitle] DELETE /users/:username/favourites/:title None A text message indicating whether the movies was successfully removed from favourites.
Unregister an existing user. /users/[Username] DELETE /users/:username None A text message indicating whether the user was successfully deleted.
